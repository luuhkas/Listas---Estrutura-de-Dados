# Projeto em C no VS Code

Projeto com exercícios em linguagem C desenvolvidos no VS Code.

## Estrutura

- `src/`: arquivos-fonte em C
- `.vscode/`: configuração do VS Code para compilar e executar
- `build/`: executáveis gerados localmente

## Tecnologias

- C
- VS Code
- Clang
- Git e GitHub

## Como executar

1. Abra o projeto no VS Code
2. Abra um arquivo `.c`
3. Use os atalhos/configurações da pasta `.vscode`

## Objetivo

Praticar lógica de programação, recursividade e estruturas de dados em C.
